# Weather App

## Features
- Provide general weather information.
- You can drag, show/hide widgets in edit mode.
- You can save widgtes arraignment in MongoDB.
- Shows your location.
- Change languages between pl/en.
- Search location.
- Add location to favourites.
- Search history.

## Technologies Used
- React with Typescript
- MongoDB
- MaterialUI
- Weather API
