## Features
- Provide general weather information.
- You can drag widgets in edit mode.
- You can save widgtes arraignment.
- Shows your location.
- Change languages between pl/en.
- Search location.
- Search history.



## Technologies Used
- React with Typescript
- MongoDB
- MaterialUI
- Weather API
